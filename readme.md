# 🚀 Report Generator

A Flask-based automation system that generates complete  reports from analyzed documents.

## 🔥 Features

- Dynamic report generation using Word templates
- Automatic extraction of:
  - Tables
  - Images (graphs)
  - Sections (3.1, 3.2, 4.1, 4.2)
- Annexure automation (A, B, C, D)
- Dynamic road table generation
- Clean and modern UI

## 🛠️ Tech Stack

- Python (Flask)
- docxtpl
- python-docx
- HTML/CSS/JAVASCRIPT
